# TaugenMotorsFrontend 
Responsive aplication for management

### Installing the dependencies 

```text
$ npm install 
```

### Clone this repository

```text
git clone git@github.com:TaugenMotors/TaugenMotorsFrontend.git
```

### Get into the folder

```text
cd TaugenMotorsFrontend 
```


### Run the app

```text
npm start 
```
